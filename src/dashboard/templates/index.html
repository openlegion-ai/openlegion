            API Reference
          
          

            
All endpoints below require an X-API-Key header. Webhook triggers use X-Webhook-Signature instead (HMAC-SHA256 of the raw body).

            

              
Credential Vault

              

                POST/mesh/credentialsStore a secret. Body: {"name":"...","value":"..."} → returns $CRED{name} handle
                DELETE/mesh/credentials/{name}Remove a secret by name
                GET/mesh/credentialsList credential names (never values)
                GET/mesh/credentials/{name}/existsCheck if a credential exists
              

            

            

              
Agent Status

              

                GET/mesh/agents/{id}/ext-statusHealth, queue depth, and budget for an agent
              

            

            

              
Webhooks

              

                POST/webhook/hook/{hook_id}Trigger an agent workflow. Requires X-Webhook-Signature header (no API key needed)
